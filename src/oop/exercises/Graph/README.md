# Graph

Implementation of the `oop.exercises.graph.Graph.java` interface. The implementation provides two type of graph:

1. `oop.exercises.graph.OrientedGraph.java`
2. `oop.exercises.graph.NonOrientedGraph.java`

`oop.exercises.graph.UseGraph.java` is the class used to test the `oop.exercises.graph.Graph.java` implementation.
